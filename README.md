# 🍷 Análise de Qualidade de Vinhos

## 📌 Sobre o projeto

Este projeto tem como objetivo analisar os fatores que influenciam a qualidade de vinhos tintos portugueses, utilizando técnicas de análise de dados com Python.

## 🎯 Objetivo

Identificar quais variáveis impactam diretamente a qualidade do vinho e gerar insights relevantes para tomada de decisão.

## 🛠 Tecnologias utilizadas

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## 📊 Análises realizadas

* Limpeza e tratamento de dados
* Análise exploratória (EDA)
* Visualização de dados
* Análise de correlação

## 📈 Principais insights

* Vinhos com maior teor alcoólico tendem a ter melhor qualidade
* Alta acidez volátil está associada a baixa qualidade
* Existe relação negativa entre acidez e qualidade

## 🧾 Conclusão

A análise mostrou que fatores como teor alcoólico e acidez volátil são determinantes para a qualidade dos vinhos, podendo auxiliar em decisões estratégicas no setor.
